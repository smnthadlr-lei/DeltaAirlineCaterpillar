Delta Air Lines – MDSC Indexing Process

Standard workflow for indexing all Delta Air Lines–related documents in the Document Routing System (DORS), focusing on the MDSC Indexing queue to ensure accuracy, consistency, and compliance.

1. Purpose

This document defines the standard workflow for indexing all Delta Air Lines–related documents in the Document Routing System (DORS) particularly MDSC Indexing queue. It ensures accuracy, consistency, and compliance with Delta’s operational, contractual, and regulatory requirements.

2. Scope

This process applies to all Document Processing Specialists handling Delta Air Lines’ incoming and outgoing documentation. This includes:

  • Physical mail received at MDSC
  • Email attachments sent to Delta/MDSC inboxes
  • Faxes received via MDSC fax lines
  • Outgoing communications requiring indexing for record retention

3. Systems used

  • DORS: Used to view documents and assign indexing details (Business Unit, Document Type, Queue).
  • MDSC Indexing queue: Used to input required indexing fields, attach documents, and finalize entries.

4. Document sources

Source Description
US Mail Physical documents delivered to MDSC mailroom.
Email Digital documents received in designated MDSC email queues.
Fax Scanned documents received via MDSC fax lines.

5. General indexing workflow

Step 1 – Document receipt

US Mail

  • Collect mail daily from the MDSC mailroom.
  • Date-stamp each document upon receipt.
  • Scan physical documents into the secure processing folder.

Email

  • Access the Delta/MDSC inbound email queue.
  • Save attachments into the secure processing folder.
  • Ensure filenames are clear and referenceable.

Fax

  • Retrieve faxes from the fax inbox.
  • Save scanned copies to the secure processing folder.

Step 2 – Document review

  • Check document quality: must be complete, legible, and relevant to Delta Air Lines operations.
  • Identify document category (e.g., benefits, payroll, employment).
  • Gather and group supporting documents for the same case.
  • If information is missing or unclear, reject the document.

Step 3 – Indexing in DORS

  • Open DORS and locate the document.
  • Identify the correct:
    • Business Unit (e.g., Benefits, Employment, Payroll)
    • Document Type (e.g., Beneficiary Form, Direct Deposit Authorization)
    • Queue (e.g., MDSC-Adoption Reimbursement, RET-Retirement)
  • Use the provided DORS mapping guide to ensure accuracy.
  • Save and route to the correct queue.

Step 4 – Quality check

  • Conduct daily peer review of indexed items.
  • Verify:
    • Correct document type
    • Accurate member ID
    • Proper queue selection in DORS
    • Legible, attached file

6. Retention & compliance

  • Follow Delta’s records retention policy.
  • Handle all sensitive documents in compliance with data privacy regulations.
  • Do not store documents outside of the approved systems.

7. Reference – Using DORS mapping

To find correct indexing details:

  • In DORS, search by document name or description.
  • Refer to the DORS Mapping Guide for:
    • Business Unit
    • Document Type
    • Queue Name
  • Double-check spelling and category—misrouted documents delay processing.

8. DORS access guide

  1. Log in to DORS: https://delta.itocservices.com/DORS/main.aspx#/home
  2. Open Workflow: Access WORKFLOW on the left panel.
  3. Select General: In Workflow, select General to view the Queues.
  4. Find MDSC-Index: Locate and open MDSC-Index.
  5. Sort DCNs: Arrange the DCNs from Oldest to Newest.
Important: Always index the old DCNs first.

Work the DCN: Click on the DCN you want to work on and fill out the information needed.
