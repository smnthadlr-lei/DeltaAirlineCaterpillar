Adding Spouse (Within 60 days of marriage event)

Within 60 Days
  • Use create record
  • Start date is marriage date
  • Review if the EE checked the indicator to retain TC pass benefits.
  • If the EE wanted to retain TC, please put a HW indicator on the SP record.
  • If EE doesn't want to retain TC, revoke the TC a day prior to marriage date.
  • Use Copy icon to update marital status, start date is process date.
  • If the EE is requesting to enroll spouse to MD using marriage event, create a marriage event in AIS.
  • Event date in AIS should be the marriage date and then route the DCN to EDM - SAP Second Day Queue.
  • Make sure to go back to the DCN after 24-48 hours to complete the enrollment using the marriage event.
  • DORS Notes = Added SP ID# (first name) in SAP eff (marriage date). Thank you
  • DORS Notes (marriage) = Routing DCN to SAP second day. Will enroll SP ID# (first name) to medical and dental coverage using marriage event once information feeds in AIS.
  • DORS Notes = Added SP ID# (first name) in SAP eff (marriage date). Thank you

Adding Spouse (Outside 60 days of marriage event)

Outside 60 Days
  • Use create icon
  • Start date is process date.
  • Review if the EE checked the indicator to retain TC pass benefits. If the EE wanted to retain TC, please put a HW indicator on the SP record.
  • If EE doesn't want to retain TC, revoke the TC a day prior to process date.
  • Use copy to update marital status, start date is process date.
  • If the EE is requesting to enroll SP to medical/dental, send a denial letter.
  • Create a denial letter, file name format: PPR Denial Letter.
  • Upload the denial to the attachment section on the DCN.
  • Send the denial letter using email or by mail if no email address is on file.
  • DORS Notes = Added SP ID# (first name) in SAP eff (process date). Thank you
  • DORS Notes (Denial) = Added SP ID# (first name) in SAP eff (process date). Sent Denial Letter
  • Regarding EE's request (EE is requesting to enroll spouse to medical and dental using marriage event, however it is already past 60 days from the marriage date). Thank you
  • Updated EE's marital status from M to S in SAP eff (process). Thank you.

  • DORS Notes (Marriage) = Routing DCN to future queue. Will add SP ID# (first name) to medical and dental using marriage event once information feeds to AIS.
  • Updated TC ID# (First name) to SP in SAP eff (marriage date). Thank you

Removing spouse due to divorce

  • Use pencil to revoke spouse, a day prior divorce date. Add spouse address if there is one indicated on the form.
  • Add divorce date on the separation date box in SAP.
  • Use copy to update marital status, start date is process date.
  • Check AIS to see if SP still has HW benefits. If there is, report a divorce event in AIS effective the date of divorce and remove SP from HW benefits.
  • DORS Notes (HW) = Revoked SP ID# (first name) due to divorce in SAP eff (a day prior divorce date). Updated EE's marital status from M to D in SAP eff (process date). Reported divorce event in AIS and removed SP ID# (first name) from (type of coverage) eff (divorce date). Thank you.

Reporting Death of Spouse

  • Use pencil to report death of SP. End date is date of death.
  • Put date of death in the box of "Since".
  • Use copy to update the EE's marital status to widowed. Start date is process date.
  • If the spouse has any HW benefits, report a death of spouse event in AIS. Event date is date of death.

  • DORS Notes = Reported SP (Spouse’s first name) death in SAP effective (date of death). Updated EE’s marital status from Married to Widowed effective (process date). Reported Death event in AIS effective (spouse’s date of death).

Adding Domestic Partner

  • Affidavit of Domestic Partnership form.
  • Affidavit must have the same address as the address on file.
  • Affidavit must be notarized within the last 60 days.
  • Date partnership commenced on the FSCF and Affidavit must be the same.
  • Proof of Domestic Partnership:
    • Lease Agreement
    • Utility bills
    • Joint bank account statement
    • Designation of the domestic partner as durable power of attorney
  • Proof of domestic partnership must show both EE and DP names, address on file, and should be backdated 6 months or older.
  • Lease agreement must show the Landlord's signature.
  • Use create icon, start date is process date.
  • Use copy to update marital status, start date is process date and make sure to add the commencement date on the "since" field.
  • DORS Notes = Added DP ID# (first name) in SAP eff (process date). Updated EE's marital status from S to DP in SAP eff (process date). Thank you.

How to Index a document

Guide
  • Step 1: Go to your assigned queue
  • Step 2: Open an item (DCN) – always choose the one on the top.
  • Step 3: Index the document.
  • Step 4: Click the double paper icon near the DCN, to copy the DCN as your productivity.
  • Step 5: Check the DCN Mapping to see if it's already in the correct queue. Click if needed to Autoroute/Complete.

Employee Name Change

  • Review the employee’s request to determine whether it is approved.
  • Supporting Document Requirements
    • Ground or Retired Employee: Government-issued ID, signed copy of passport, or court order for name change.
    • Flight Attendant or Pilot Employee:Signed copy of passport.
  • Note:Ensure that the information on the supporting document matches the details in the Employee Information Update Form or self-service.
  • Check the employee’s group code.
  • Navigate to SAP > PA30 > Select Personal Data (Infotype 0002) > Click the Copy Paper icon, enter the employee’s updated name, then save the changes by clicking the Disk icon.
  • DORS Notes: Updated the employee’s name in SAP, effective (process date).

Employee Address Change

  • Review the employee’s request to determine whether it is approved.
  • If approved, proceed with updating the employee’s address.
  • Note:If the Employee Update Form only includes the home address, the mailing address should also be updated to match the home address. Additionally, do not backdate the start date in SAP for address change requests.
  • Navigate to SAP > PA30 > Select Address (Infotype 0006) > Click the Copy Paper icon, then enter the employee’s updated address > Then save the changes by clicking the Disk icon.
  • Double-check the update by switching to PA20.
  • DORS Notes: Updated the employee’s home and mailing address in SAP, effective (process date).

Voluntary Spouse Removal (Divorce)

  • 1. Find employee in SAP
  • 2. Find spouse documented on form on IT21.
  • 3. Confirm the spouse listed on the form has been voluntarily removed (voluntarily removed date on IT21 will be populated and the end date will be 1 day before the voluntarily removed date.)
  • 4. Using the Pencil Icon
    • Update the “End-Date” field to reflect the date before the divorce date (end date will be 1 day before the divorce date).
    • Update the spouse’s record with the divorce date.
    • Save the changes
  • 5. If there is another Spouse or Domestic Partner on file – this is ok. However, the existance of a new spouse or domestic partner will mean that there will be special processing of IT2 for the marital status.
    • If there is no other spouse on file, proceed to step #7
    • If there is a recent spouse or domestic parnter on file, proceed to step #13
  • 6. Go to IT2
  • 7. If there was no other spouse on file
  • 8. Copy the top record on IT2
  • 9. Update the Start Date to today’s date.
  • 10. Change the employee's Marital Status to Divorced.
  • 11. Change the Marital Status Since Date to the date of the Divorce.
  • 12. Save the changes by clicking the Disk Icon
  • 13. If there was another spouse or domestic partner the assumption is the most current record on IT2 is now associated with the new spouse or domestic partner. The agents must confirm this, by reviewing IT21 for a recent add of a spouse or domestic partner.
  • 14. If all assumptions are correct the agent will need to insert a record before the most current record. The best way to do this is to find the record before the current record and select copy.
  • 15. The start and end dates of the record can both be equal to the day before the current record.
  • 16. The inserted record will have a Marital Status of Divorced.
  • 17. The inserted record will have a Marital Status Since Date equal to the date of divorce.
  • 18. Save the changes
